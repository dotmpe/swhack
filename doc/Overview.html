Semantic Web Tutorial using N3

This is an introduction to semantic web ideas aimed at someone with experience in programming, perhaps with web sites and scripting, who wants to understand how RDF is useful in practice. The aim is to give a feel forwhat the Semantic Web is and how it will be useful in practice. This is illustrated using the N3 language, which is easy to read and write, and cwm which is an experimental general purpose program for semantic web stuff.

The tutorial is in the making: places linked below have text. This material will be presented at the WWW2003 in Budapest, 2003-05
